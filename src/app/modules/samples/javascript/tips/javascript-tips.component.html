Javascript tips ...

1-Usar el poperador de doble negación !! para convertir cualquier valor a booleano para sin tener que usar ninguna otra función en javascript validar si esta varieble tiene algún valor, ejemplo: !this.isValue = !!value; retorna false si es cero, null, undefined o NaN y true en el resto de lo casos.

2-Cómo convertir un string a un número: lo que queremos es transformar el texto a número, para ello retornamos el string con el signo de adicción delante: +numString y retorna el número.

Podemos hacer un condicional corto cuando verificamos el valor de un método booleano y tenemos que llamar a otro método en caso verdadero en ves de realizar otra condicional se puede hacer todo en una línea: conectado && this.loaddata();

Cuando pasamos parámetros a un método y queremos que tenga valores por defecto podemos hacerlo de esta forma:
this.nameAnime = nameAnime || 'Myname-default';
