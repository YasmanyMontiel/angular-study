Funciones para trabajar con arreglos forEach(), reverse(), for()

El método forEach() ejecuta la función indicada una vez por cada elemento del array. Este método puede ayudarte a recorrer los elementos de un array.

En javascript puedes usar reverse(). El mismo invierte el orden de los elementos.


En javascript puedes usar for(). Y como límite para parar: arreglo.length - 1 e ir decrementando el ciclo. El mismo invierte el orden de los elementos.
